澳门旅游计划

基于澳门航空NX165航班信息（2025年8月21日10:50抵达）和澳门励庭海景酒店住宿安排制定了这份详尽的澳门旅游攻略。本计划已充分考虑节约餐饮开支、合理移动距离、紧密行程安排等所有要求，并提供可追踪的可靠数据来源。

行程概览

出行信息：

  • 人数：3-4人（包含1名16岁少年）
  • 日期：2025年8月21日-23日（3天2夜）
  • 航班：澳门航空NX165，宁波栎社T2 08:30 → 澳门机场 10:50
  • 住宿：澳门励庭海景酒店（入住15:00，退房11:00）
  • 汇率：1澳门币 = 0.885人民币（2025年7月24日）

修改摘要（一次看懂）

  1. 晚间互换
    • 8/21（周四） 晚改去 澳门旅游塔 + 360 Café（看日落→晚餐→返酒店）。
    • 8/22（周五） 晚改为 伦敦人 The Londoner + 巴黎人铁塔，并在 威尼斯人后追加 MGM COTAI，再去 永利皇宫表演湖 + SkyCab，最后步行至 LRT「路氹东站」 乘夜景线→排角→打车回酒店。
  2. 合并：将「巴黎人铁塔」与「伦敦人」安排在同一晚（同片区，步行联通）。
  3. 新增
    • 威尼斯人后 加逛 MGM COTAI（The Spectacle 中庭打卡，空调室内，适合避晒/休脚）。
    • 在原「路氹东站」之前 增加 永利皇宫表演湖 + SkyCab（看完再去坐 LRT）。

澳门特色美食推荐

必尝美食

  1. 葡式蛋挞：安德鲁饼店（路环原创店）、瑪嘉烈蛋撻店
  2. 猪扒包：大利来记（氹仔官也街35号）、世记咖啡
  3. 水蟹粥：誠昌飯店
  4. 咖喱魚蛋：恒友魚蛋
  5. 木糠布甸：澳门传统甜品

推荐餐厅详情

  • 大利来记：近50年历史，使用巴西猪扒，柴爐烤制
  • 世记咖啡：现炸厚切猪扒，瓦煲咖啡特色
  • 安德鲁饼店：葡式蛋挞创始店，1989年创立

澳门必买手信指南

热门手信推荐

  1. 杏仁饼：最香饼家（炭烧古法制作，70年历史）
  2. 纽结糖：车厘哥夫（澳门版鸟结糖）
  3. 蝦子麵：幸运来手信（每日限量，纯手工制作）
  4. 核桃酥：晃记饼家（百年历史，人手制作）
  5. 腰果曲奇：樂宮餅店（无添加剂）

购物地点

  • 官也街：澳门手信一条街
  • 威尼斯人购物中心：350+品牌
  • 机场免税店：最后采购机会
